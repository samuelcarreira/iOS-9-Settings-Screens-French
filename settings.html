Wi-Fi

 

Caso a opção DHCP esteja ativa, o dispositivo poderá ter um endereço de IP diferente cada vez que se liga a um ponto de acesso. A opção Statique permite que o utilizador defina um endereço de IP fixo para o seu dispositivo (o utilizador terá de respeitar a gama de endereços do seu ponto de acesso, valores mais comuns: 192.168.0.x, 10.0.1.x).
A faixa de endereços reservadas para redes privadas são as seguintes:
10.0.0.0 - 10.255.255.255 | 172.16.0.0 - 172.31.255.255 | 192.168.0.0 - 192.168.255.255
O endereço do servidor de DNS tipicamente é o endereço do ponto de acesso que está associado ao ISP, no entanto o utilizador pode utilizar outro servidor de DNS público como por exemplo:
8.8.8.8 | 208.69.38.205
